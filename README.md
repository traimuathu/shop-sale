### SHOP-SALE

#### How to run :wrench:

You must install **Nodejs** and **Yarn**

Clone this repository
```shell
git clone https://github.com/traimuathu/shop-sale.git
```

```shell
cd shop-sale
```

Install dependencies
```shell
yarn install
```
and install **nodemon** (Dev dependencies)
```shell
yarn add nodemon --dev
```

Start server
```shell
yarn start
```
